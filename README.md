# 🍽️ El Rincón del Tapeo – Página Web

Proyecto final de desarrollo web estático que simula la página de un restaurante de tapas.
La web incluye navegación entre páginas, carrusel de imágenes, contenido multimedia y diseño responsive utilizando HTML y CSS.

---

## 📌 Descripción

**El Rincón del Tapeo** es una página web que representa el sitio de un restaurante ficticio especializado en tapas y raciones españolas.

El objetivo del proyecto es practicar conceptos fundamentales de **desarrollo web front-end**, incluyendo:

* Estructura semántica en HTML
* Estilado con CSS
* Uso de Flexbox
* Animaciones CSS
* Diseño responsive
* Integración de imágenes, gifs y audio

---

## 🚀 Tecnologías utilizadas

* **HTML5**
* **CSS3**
* **Flexbox**
* **Animaciones CSS**
* **Google Fonts (Mulish)**

## 🌐 Páginas de la web

La web está formada por varias secciones:

### 🏠 Home

* Página principal
* Carrusel de imágenes del restaurante
* Presentación visual del sitio

### 🍤 Nuestros Sabores

* Portfolio de platos y tapas
* Uso de imágenes y gifs para mostrar comida

### 👨‍🍳 Nosotros

* Información sobre el restaurante
* Historia o biografía

### 📩 Contacto

* Información de contacto del restaurante

## 🎨 Características destacadas

* 📱 **Diseño responsive** con `meta viewport`
* 🖼 **Carrusel de imágenes** con animaciones CSS
